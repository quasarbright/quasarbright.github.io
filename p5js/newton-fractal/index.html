Newton's Method Fractal Visualization

What is Newton's Method?

Newton's method is an numerical method for finding roots (zeros) of a function. For a function \(f(z)\) and some starting point \(z_0\), we iteratively apply the following recurrence relation:

\[z_{n+1} = z_n - \frac{f(z_n)}{f'(z_n)}\]

This process brings the point closer to \(f(z) = 0\).

How This Visualization Works

Each pixel on the screen represents a \(z_0\) point in the complex plane. We apply Newton's method to each point and color it based on which root it converges to, or black if it doesn't converge.

The polynomial \(f(z)\) being visualized is in the form:

\[f(z) = (z-r_1)(z-r_2)(z-r_3)\ldots(z-r_n)\]

Where \(r_1\), \(r_2\), etc. are the roots represented by draggable circles. You can add, remove, and drag roots to change the polynomial and see how the fractal changes in real-time.

Interactive Features

  • Pan: Click and drag to move around
  • Zoom: Use mouse wheel or pinch gestures
  • Modify Roots: Drag the colored circles to change the polynomial
  • Add/Remove Roots: Use the buttons to add or remove roots
  • Reset: Reset the view or roots to their default state
  • Mandelbrot Mode: Each pixel becomes a new root, and the color is determined by which root the average of all roots (including the pixel) converges to

Mandelbrot Mode

In standard mode, we fix a polynomial \(f(z) = (z-r_1)(z-r_2)...(z-r_n)\) and for each pixel \(z_0\) in the complex plane, we run Newton's method to see which root it converges to.

In Mandelbrot mode, each pixel \(z_0\) becomes a new root of the polynomial. For each pixel:

  1. We treat the pixel's position \(z_0\) as a new root
  2. We calculate the average of all roots (including \(z_0\))
  3. We run Newton's method on this average position
  4. We color the pixel based on which root it converges to (or black if it doesn't converge)

This creates a different kind of fractal analogous to the mandelbrot set fractal, whereas standard mode is analogous to the julia set fractal. In fact, you can get the mandelbrot set to appear by reducing down to 2 roots and looking for black pixels.
