Complex Function Visualization

Enter GLSL code that returns a vec2 result. Your function should use the variables z and c which are both vec2 (complex numbers).

Available complex functions:

  • csquare(z) : z²
  • ccube(z) : z³
  • cpow(z, n) : z^n
  • cmul(a, b) : a * b
  • cdiv(a, b) : a / b
  • csin(z), ccos(z)
  • cexp(z), clog(z)

Custom Parameters

You can add custom parameters to use in your function:

  1. Go to the Parameters tab
  2. Enter a name (e.g., "p"), real and imaginary values
  3. Click Add
  4. Use the parameter in your function directly by its name (e.g., if you named your parameter "p", just use p in your function)

For each parameter, you can choose between three modes:

  • Fixed: Uses the specified constant value
  • Dot: Allows you to drag a colored dot to set the value
  • Pixel: Uses the current pixel position (like the Mandelbrot set)

Example: If you create a parameter named "p", refer to it simply as p in your function:

return csquare(z) + cmul(c, p);
